# AI Diplomacy Asset Naming Registry

Для окна `aidip_dialog` используется единый префиксный стиль имен:

- `ai_dialog_*` — базовые элементы окна, вкладки, кнопки действий, +/- контролы.
- `ai_chronicle_*` — журнал контактов, фильтры истории, служебные панели хронологии.
- `ai_secret_*` — скрытые/разведывательные элементы и маркеры секретных факторов.

## Зарезервированные имена

### ai_dialog_*
- `ai_dialog_tab_idle`
- `ai_dialog_toggle`
- `ai_dialog_plus`
- `ai_dialog_minus`
- `ai_dialog_send`

### ai_chronicle_*
- `ai_chronicle_tab_idle`
- `ai_chronicle_panel`

### ai_secret_*
- `ai_secret_tab_idle`
- `ai_secret_toggle`
