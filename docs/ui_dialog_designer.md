# UI Dialog Designer: AI Diplomacy Window

Этот документ фиксирует структуру окна `aidip_dialog` и ожидаемое поведение блоков.

## 1) Карточка страны
- Флаг, название страны, идеология, текущий лидер.
- Краткий дипломатический статус (война/мир/нейтралитет).
- Индикатор базового отношения (например, `-100..+100`).

## 2) Видимые/скрытые факторы отношения
- **Видимые факторы**: общие границы, торговля, гарантии, фокусные эффекты.
- **Скрытые факторы**: секретные цели, скрытые модификаторы ИИ, внутренние приоритеты.
- Переключатель отображения секретных факторов (только при наличии разведданных).

## 3) Тип обращения
- Выпадающий список типа обращения:
  - Дружественное предложение,
  - Давление,
  - Ультиматум,
  - Запрос отсрочки,
  - Контрпредложение.
- Подсказки по допустимым типам в текущем контексте.

## 4) Параметры сделки
- Секция параметров по категориям:
  - ресурсы,
  - территории,
  - военные обязательства,
  - гарантии/пакты,
  - разведданные.
- Для каждого параметра — переключатель включения и поле величины/уровня.

## 5) Цена/стоимость
- Плашка «Стоимость предложения» (политическая, экономическая, военная).
- Блок «Оценка вероятности принятия» с цветовой индикацией.
- Прогноз штрафов за завышенные требования.

## 6) История контактов
- Хронологический журнал событий переговоров.
- Фильтры: все / инициативы / ответы / скрытые события.
- Кнопка быстрого перехода к последнему ответу.

## 7) Ответ/контрпредложение
- Отдельная панель входящего ответа:
  - принять,
  - отклонить,
  - выставить контрпредложение,
  - выдвинуть ультиматум,
  - запросить задержку.
- Поле комментария ИИ (видимое игроку резюме причин).

## 8) Кнопка отправки
- Основная кнопка `Отправить`.
- Перед отправкой — валидация обязательных параметров.
- При ошибке — список причин блокировки отправки.
