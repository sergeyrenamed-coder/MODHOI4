# Diplomacy State Model (AI DIP)

Документ описывает параметры состояния дипломатических отношений между двумя странами (`A -> B`).

## Общие правила

- Базовый диапазон большинства числовых параметров: `[-100; +100]`.
- Нейтральная точка: `0`.
- Положительные значения усиливают кооперацию, отрицательные — конфликтность (если не указано иное).
- Изменения применяются через события-источники (`сделка`, `отказ`, `угроза`, `война`, `помощь`) и далее затухают по заданной модели.

## Параметры

| Параметр | Диапазон | Скорость изменения (типично за событие) | Затухание со временем | Источники изменения |
|---|---:|---:|---|---|
| `trust` | `[-100; +100]` | `±2..±12` | `1.5%/мес` к `0` | Сделка `+`, Отказ `-`, Угроза `-`, Война `--`, Помощь `++` |
| `fear` | `[0; 100]` | `±3..±15` | `3%/мес` к `0` | Сделка `-` (если гарантии), Отказ `+` (при ультиматумах), Угроза `++`, Война `++`, Помощь `-` |
| `respect` | `[-100; +100]` | `±2..±10` | `1%/мес` к `0` | Сделка `+` (выгодная/честная), Отказ `-` (нарушение статуса), Угроза `±` (успешная `+`, пустая `-`), Война `±`, Помощь `+` |
| `dependence` | `[0; 100]` | `±1..±8` | `2%/мес` к `0` | Сделка `+` (ресурсная привязка), Отказ `-`, Угроза `+` (принуждение), Война `±`, Помощь `+` |
| `leverage` | `[0; 100]` | `±2..±12` | `4%/мес` к `0` | Сделка `+` (асимметричная), Отказ `-`, Угроза `+`, Война `+` (при военном превосходстве), Помощь `+` |
| `influence` | `[0; 100]` | `±1..±7` | `2.5%/мес` к `0` | Сделка `+`, Отказ `-`, Угроза `±`, Война `±`, Помощь `++` |
| `ideological_affinity` | `[-100; +100]` | `±1..±6` | `0.5%/мес` к `0` | Сделка `+` (блоковая), Отказ `-`, Угроза `-`, Война `--`, Помощь `+` |
| `threat_perception` | `[0; 100]` | `±4..±20` | `5%/мес` к `0` | Сделка `-`, Отказ `+` (жёсткий), Угроза `++`, Война `+++`, Помощь `-` |
| `gratitude` | `[0; 100]` | `±5..±20` | `8%/мес` к `0` | Сделка `+` (уступка), Отказ `-`, Угроза `-`, Война `--`, Помощь `+++` |
| `betrayal_memory` | `[0; 100]` | `+5..+25` / `-0..-3` | `1%/мес` к `0` | Сделка `-` (серия соблюдённых), Отказ `+` (нарушение обещаний), Угроза `+`, Война `++` (особенно после пактов), Помощь `-` |
| `reliability_score` | `[0; 100]` | `±2..±10` | `1.5%/мес` к `50` | Сделка `+` (исполнение), Отказ `-` (срыв договорённостей), Угроза `-`, Война `-` (если агрессор), Помощь `+` |
| `opportunism_score` | `[0; 100]` | `±2..±12` | `2%/мес` к `50` | Сделка `-` (долгие обязательства), Отказ `+` (выбор выгоды), Угроза `+`, Война `+` (удар в слабость), Помощь `-` |
| `public_stance` | `[-100; +100]` | `±3..±18` | `4%/мес` к `0` | Сделка `+`, Отказ `-`, Угроза `-`, Война `--`, Помощь `+` |
| `hidden_stance` | `[-100; +100]` | `±1..±10` | `1%/мес` к `0` | Сделка `±` (по реальной выгоде), Отказ `-`, Угроза `-`, Война `--`, Помощь `+` |

## Примечания по источникам

- `сделка` — торговля, пакты, гарантии, раздел сфер, лицензии.
- `отказ` — отклонение запроса, срыв обещаний, неисполнение обязательств.
- `угроза` — дипломатический ультиматум, демонстрация силы, политическое давление.
- `война` — объявление войны, аннексия, оккупация, отдельный мир.
- `помощь` — ленд-лиз, экспедиционный корпус, добровольцы, экономическая поддержка.

## Рекомендуемые агрегаты для логики

- **Кооперативность**: `trust + respect + influence + ideological_affinity + gratitude`.
- **Подчиняемость**: `fear + dependence + leverage`.
- **Риск предательства**: `opportunism_score + betrayal_memory - reliability_score`.
- **Внешне/внутренне расхождение**: `abs(public_stance - hidden_stance)`.
